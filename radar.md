# radar

for using the radar, you have two options:

## 1. using an already-hosted radar server

if you have an address of a server, just type it in, and it should connect

Public server instances:

1. <http://radar.disphost.cc:80> (discord: asfgarasd)

## 2. self-host

1. clone the repo, as usual
2. make sure you have some JavaScript runtime installed (nodejs, deno, bun)
3. the `run_radar.sh` script should take care of launching the server

note: https/ssl is not supported (annoying to set up),
but might get support later.
